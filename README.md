# Study Timer

This is a simple study timer made with HTML, CSS, and JavaScript. It lets you enter a number of minutes and count down while you work. You can start, pause, and reset the timer at any time.

---

## Features
- Start, pause, and reset the timer  
- Progress bar that fills up as time runs out  
- Sound plays when the timer finishes  
- Dark mode button  
- Timer does NOT restart when paused

---

## Accessibility Features
- Label added to the minutes input  
- ARIA live region so screen readers read the timer  
- Buttons include aria-labels for clearer meaning

---

## How to Use
1. Open https://mathew-o-2026.github.io/study-timer/ in your browser  
2. Type how many minutes you want  
3. Click **Start**  
4. Click **Pause** to pause and **Start** again to resume  
5. Add a file named `alarm.mp3` to the folder for the sound
